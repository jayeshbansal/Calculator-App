## 🧮 Calculator Web App (HTML, CSS & JavaScript)

A responsive **Calculator Web Application** built using **HTML5, CSS3, and JavaScript**. The project performs basic arithmetic operations with a clean, user-friendly interface while implementing input validation and error handling to improve the overall user experience.

### 🚀 Key Features

* Responsive design for Desktop and Mobile devices.
* Perform basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)
  * Percentage (%)
* Real-time expression display.
* Clear (**C**) button to reset calculations.
* Delete (**DEL**) button to remove the last entered character.
* Prevents entering consecutive operators.
* Handles invalid expressions with error messages.
* Clean and modern calculator UI.
* Optimized using JavaScript event handling.

### 🛠️ Technologies Used

* JavaScript (ES6)
